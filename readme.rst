************
Installation
************

1. Install XAMPP
2. Download Sebagai Zip gitHub https://github.com/MrAufa111/Kasir-Codeigniter
3. Pindahkan File Ke dalam file C:xampp/htdocs
4. Buka Browser dan Jalan kan localhost (localhost/kasir-codeigniter)
